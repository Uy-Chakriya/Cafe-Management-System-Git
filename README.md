# ☕️ Cafe Management System  

## 📖 Overview  
The **Cafe Management System** is a desktop application built with **JavaFX** and **Maven**, designed to help cafes streamline daily operations. It offers modules for authentication, customer management, menu handling, and order processing, along with an administrative dashboard for central control.  

This project is suitable for **small to medium-sized cafes** looking for a digital solution to replace manual management processes.  

---

## ✨ Features  

- 🔐 **User Authentication** – Secure login (`login.fxml`, `LoginController.java`)  
- 👥 **Customer Management** – Add, update, and delete customers (`customers.fxml`, `CustomerController.java`)  
- 🍔 **Menu Management** – Manage menu items, categories, and prices (`menu.fxml`, `MenuController.java`)  
- 🧾 **Order Management** – Create and process orders (`order.fxml`, `OrderController.java`)  
- 📊 **Admin Dashboard** – Centralized dashboard for management (`admin.fxml`, `AdminController.java`)  

---

## 🛠 Tech Stack  

- **Language**: Java 21  
- **GUI Framework**: JavaFX 21  
- **Build Tool**: Maven 3.8.5+  
- **IDE Support**: IntelliJ IDEA, NetBeans, Eclipse (with JavaFX plugin)  

---

## 🚀 Getting Started  

### ✅ Prerequisites  
Make sure you have:  
- **JDK 21 or higher** installed  
- **Maven 3.8.5+** installed (or use included Maven Wrapper `mvnw`)  
- JavaFX SDK (if running manually without Maven plugin)
